# ampy Cheatsheet
[official doc](https://cdn-learn.adafruit.com/downloads/pdf/micropython-basics-load-files-and-run-code.pdf)

### Run python script
`ampy -p $AMPY_PORT run`

### Copy script to the MSP
`ampy -p $AMPY_PORT put`

### List files in the MSP
`ampy -p $AMPY_PORT ls`
